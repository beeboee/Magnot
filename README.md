# Magnot — Forge 1.18.2

Core Magnot port for Minecraft 1.18.2 Forge and Create 0.5.1.

Included: persistent ferrous regions, tube placement/removal, region outlines, API v2, and optional Simple Magnets / Item Collectors hooks.

This branch derives from the shared 1.18–1.19 Forge core. It does not yet include Sable, moving sub-level support, filtered-region behavior, custom region entities, or the complete 1.21.1 compatibility matrix.
