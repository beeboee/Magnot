# Magnot

Magnot adds Create-style ferrous paste regions that block supported magnet and vacuum effects from pulling item entities through protected areas.

See [magnet compatibility](docs/COMPATIBILITY.md) for supported, planned, and future-target magnet integrations.

See [the public compatibility API](docs/API.md) if you maintain a mod with magnet, vacuum, remote item collector, absorption hopper, or similar item-moving behavior and want to support Magnot directly.
