# Magnot — Forge 1.19.2

Core Magnot port for Minecraft 1.19.2 Forge and Create 0.5.1.

Included: persistent ferrous regions, tube placement/removal, region outlines, API v2, and optional Simple Magnets / Item Collectors hooks.

Not yet included: Sable, moving sub-level support, filtered-region behavior, custom region entities, or the complete 1.21.1 compatibility matrix.
