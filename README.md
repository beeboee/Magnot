# Magnot — Forge 1.16.5 alpha

Core Java 8 port for Minecraft 1.16.5 and Forge 36.2.9+.

## Requirements

- Minecraft 1.16.5
- Forge 36.2.9 or newer
- Create 0.3.2g through the 0.3.x line
- Java 8 runtime

## Alpha 2 integration coverage

Verified legacy targets retained for:

- Simple Magnets
- Item Collectors
- Mob Grinding Utils absorption hopper
- Modular Routers vacuum module
- Mekanism magnetic attraction
- Draconic Evolution magnet
- Reliquary Reincarnations fortune coin (`xreliquary` package)

Sophisticated Core and Industrial Foregoing are no longer claimed on 1.16.5 because the selected jars did not verify the alpha.1 target methods. Alpha 2 also remaps Minecraft call sites inside optional mod classes for production Forge.

Included core features are persistent ferrous regions, tube placement/removal, particle outlines, and the compatibility API. Sable, moving sub-levels, filtered regions, and custom region entities remain exclusive to newer branches.
