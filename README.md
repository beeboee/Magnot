# Magnot — Forge 1.16.5

Core Java 8 port for Minecraft 1.16.5, Forge 36.2.9+, and Create 0.3.2g+.

Included: persistent ferrous regions, tube placement/removal, particle outlines, compatibility API, and an optional Simple Magnets hook.

This is intentionally smaller than the 1.21.1 branch. It does not include Sable, moving sub-levels, filtered regions, custom region entities, or the modern compatibility matrix.
