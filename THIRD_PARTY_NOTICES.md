## Create

Magnot's native selection backend follows the interaction and presentation concepts used by Create's glue selection system. When Create is installed, Magnot calls Create/Catnip APIs directly for raycasting, outlines, fade behavior, face texturing, and sounds. Magnot does not include Create textures, models, sounds, or other Create assets.
