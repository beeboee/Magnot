# Magnot v0.1 Scope

Magnot adds Create-style ferrous paste regions that block supported magnet upgrades from pulling items through protected areas.

## Target

- Minecraft `1.21.1`
- NeoForge `21.1.230`
- Create `6.0.10`
- Java `21`

## Included first

- `Ferrous Tube` item
- `Ferrous Paste` item
- `Iron Dust` item
- Create crushing recipe: iron ingot -> iron dust
- Ferrous paste recipe: iron dust + slime adhesive
- Ferrous tube recipe using ferrous paste, iron nugget, and iron plate/sheet tags
- Server-side ferrous cuboid storage
- Tube corner selection on blocks
- Removal by attacking a selected region while holding the tube
- Shared magnet rule helper for Sophisticated Backpacks / AE2WTLib adapters

## Deliberately not included yet

- Universal magnet blocking
- Client overlay renderer
- Sophisticated Backpacks mixin
- AE2WTLib mixin
- Ponder scenes
- Permissions/claims
- Energy costs

## Build note

This repo does not include a Gradle wrapper yet. Use a local Gradle install with Java 21:

```bash
gradle build
```
